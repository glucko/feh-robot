◆ COUNTS_PER_ENCODER

const int COUNTS_PER_ENCODER = 318

Number of counts per encoder revolution for the servo motors.

This constant represents the number of counts registered by the motor encoder for one complete revolution of the shaft. Used for accurate position and movement distance calculations.

Note
The value of 318 counts is specific to the motor encoders used in the FEH robot

Threshold value for detecting absence of light.

When the CdS cell reading exceeds this value, the robot will identify it as no light present.

◆ redThreshold
